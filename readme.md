# Лабораторная работа: Работа с Docker и Web-сервером Apache

## Название лабораторной работы
Запуск контейнера Ubuntu с установкой Web-сервера Apache и отображением страницы "Hello, World!"

---

## Цель работы
- Освоить базовые команды ОС Debian/Ubuntu.  
- Познакомиться с Docker и его основными командами.  
- Научиться запускать контейнеры и устанавливать внутри них сервисы, такие как Apache.  

---

## Задание
1. Запустить контейнер Ubuntu.  
2. Установить Web-сервер Apache.  
3. Создать веб-страницу с текстом **"Hello, World!"**.  
4. Проверить работу сервера в браузере.  
5. Ознакомиться с конфигурацией Apache.  
6. Завершить работу с контейнером и удалить его.  

---

## Описание выполнения работы

### 1. Создание проекта
```bash
mkdir containers04
cd containers04
touch README.md
````

Создана папка проекта и файл README.md для отчёта.

---

### 2. Запуск контейнера Ubuntu

```bash
docker run -ti -p 8000:80 --name containers04 ubuntu bash
```

* `docker run` — запускает контейнер.
* `-ti` — интерактивный терминал.
* `-p 8000:80` — проброс порта 80 контейнера на порт 8000 локального компьютера.
* `--name containers04` — имя контейнера.
* `ubuntu bash` — образ Ubuntu с запуском bash.

**Результат:** открылся терминал контейнера Ubuntu.

---

### 3. Обновление пакетов

```bash
apt update
```

* Обновляется список пакетов и доступных версий.

---

### 4. Установка Apache

```bash
apt install apache2 -y
```

* Устанавливается веб-сервер Apache.
* `-y` подтверждает установку автоматически.

---

### 5. Запуск сервера

```bash
service apache2 start
```

* Запуск веб-сервера Apache внутри контейнера.
* Сервер слушает порт 80.

**Проверка:**
В браузере перейти по адресу `http://localhost:8000`.

### 6. Просмотр файлов веб-сервера

```bash
ls -l /var/www/html/
```

* Просмотр содержимого директории с веб-страницами Apache.

---

### 7. Создание страницы "Hello, World!"

```bash
echo '<h1>Hello, World!</h1>' > /var/www/html/index.html
```

* Создан новый файл `index.html` с текстом **Hello, World!**
* Старый файл перезаписан.

**Проверка:** обновить страницу `http://localhost:8000` — теперь отображается **Hello, World!**

### 9. Завершение работы с контейнером

```bash
exit
```

* Закрыли терминал контейнера.
* Контейнер остановлен.

---

### 10. Просмотр и удаление контейнера

```bash
docker ps -a
docker rm containers04
```

* `docker ps -a` показывает все контейнеры.
* `docker rm containers04` удаляет контейнер с диска.
